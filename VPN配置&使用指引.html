不同系统使用不同的客户端。请根据你的设备下载并安装对应软件，安装完成后再进入第三步做配置。

WindowsClash Verge Rev

前往 GitHub 项目的 Releases 页面，下载最新版 Windows 安装包（.exe）并安装。

⬇ 官方项目地址（Releases 中选 Windows 版本） https://github.com/clash-verge-rev/clash-verge-rev

macOSClash Verge Rev

同一 GitHub 项目，在 Releases 中选择 macOS 版本（区分 Intel 芯片 x64 与 Apple Silicon aarch64）下载安装。

⬇ 官方项目地址（Releases 中选 macOS 版本） https://github.com/clash-verge-rev/clash-verge-rev

iOSShadowrocket（小火箭）

  • 需将 App Store 账号切换为 境外区（如美区）；
  • 在 App Store 搜索 「Shadowrocket」 并安装；
  • 该 App 为 付费应用，首次购买约 2.99 美元，购买后与 Apple ID 绑定，后续可终身使用。
💡 提示 国区 App Store 搜不到 Shadowrocket，必须先切换到已购买过该应用或支持外区支付的境外 Apple ID。下面提供完整的切换方法。

三

配置与使用

导入订阅 → 选择节点 → 开启代理（各平台图文步骤）
收起展开▾

软件安装完成后，把第一步复制的「订阅链接」导入软件，再选择节点并开启代理即可上网。请点击下方你所用的系统展开对应教程（默认收起，只看自己的平台即可）。

WindowsClash for Windows 一键订阅▾

1准备工作：关闭系统代理

打开软件进入 General（常规/主页） 页面，确保 System Proxy（系统代理） 和 TUN 模式 开关都处于 关闭（OFF） 状态，以免影响导入。

🖼️点击查看截图：关闭系统代理收起截图▾
确保「系统代理」「TUN 模式」为关闭状态

2官网操作：一键导入

登录机场官网 → 左侧菜单 订阅中心 → 在 Windows / Clash 卡片下点击 【一键导入】 按钮。若浏览器弹出「是否打开 Clash for Windows」，点击 打开/允许。

🖼️点击查看截图：一键导入收起截图▾
官网订阅中心点击「一键导入」，软件自动跳转到「配置」页并出现绿色配置卡片即导入成功
💡 无法一键导入？点击按钮无反应时，可在官网选择 复制订阅地址，然后在软件 Profiles（配置） 页顶部输入框粘贴 URL 手动下载。

3选择节点并开启代理

进入 Proxies（代理） 页面，顶部模式建议选择 Rule（规则），在节点列表中选择要使用的节点。最后回到主页打开 系统代理 开关，即可用浏览器访问 Google 等网站测试。

🖼️点击查看截图：开启系统代理收起截图▾
完成导入后打开「系统代理」开关即可正常上网

1填入订阅链接

打开 Clash Verge Rev，点击左侧 「订阅」 栏，将复制的订阅链接粘贴到顶部输入框，点击 「导入」。列表中出现配置文件即表示导入成功。

2开启系统代理

进入左侧 「设置」，找到 「系统代理」 并打开开关，此时即可正常访问网络。

打开「系统代理」开关（正常导入配置后即可使用）

3选择节点（代理组）

点击左侧 「代理」，模式保持在 「规则」，展开节点列表，选择一个需要的地区（如香港或日本）。可点击闪电图标测试节点延迟，有数值代表节点可用。

🖼️点击查看截图：选择节点收起截图▾
深色表示当前选中节点；有延迟数值代表节点正常可用

⚠️ Mac 无法上网 / 系统代理打开无效？ 若点击「系统代理」后自动关闭或无反应：进入 设置 页面，点击 「虚拟网卡模式（TUN）」 旁的小扳手图标进行安装/修复，然后开启 TUN 模式 开关，通常可解决 Mac 系统的权限冲突问题。

四

更新订阅

节点超时或收到维护通知时，刷新订阅同步最新服务器
收起展开▾

当遇到节点超时、新增节点刷不出来，或收到「维护通知」时，请务必更新订阅，以同步最新的服务器配置。

通用原则

打开主/备官网，重新登录账号，获取最新的订阅 URL 后，重新添加/更新配置。大多数客户端支持在软件内直接点「刷新」按钮更新，无需重新粘贴链接。

Windows

  1. 打开软件，进入左侧 Profiles（配置）；
  2. 找到你的订阅卡片（通常显示为 WgetCloud）；
  3. 右键点击 卡片 → 选择 Update（更新），或直接点击卡片右上角的 🔄 刷新图标；
  4. 右下角弹出绿色提示、卡片时间变为「Just now」即更新成功。

macOS

  1. 打开 Clash Verge，进入左侧 「订阅」；
  2. 在订阅卡片上点击 🔄 刷新按钮；
  3. 卡片显示新的更新时间即更新成功。
🖼️点击查看截图：Windows 更新订阅收起截图▾
Windows：在「配置」页右键卡片选 Update 或点击刷新图标
🖼️点击查看截图：macOS 更新订阅收起截图▾
macOS：在「订阅」页点击卡片上的刷新按钮

iOS

在 Shadowrocket 首页，下拉刷新，或进入订阅配置点击更新按钮同步节点。

Android 鸿蒙

进入「配置」页，点击订阅卡片右侧的刷新/更新按钮同步最新节点。

❌ 更新失败怎么办？ 若点击更新后提示红色 Error 或无反应，请依次尝试：
1. 暂时 关闭「系统代理」 开关后再重试；
2. 检查电脑/手机 时间是否准确；
3. 若依然失败，请到官网 复制官网订阅链接，重新导入配置。

常见问题排查

无法上网时的快速定位清单 + 合规提醒
收起展开▾

遇到无法上网时，点击下面对应的问题查看解决方法。

❓ 软件已开，但打不开网页▾
检查是否已打开「系统代理」开关；确认已选中一个有延迟数值的可用节点（延迟显示为数字而非超时）。
❓ 节点全部超时 / 显示红色▾
先更新订阅（见第四步）同步最新服务器；仍不行则切换到备用线路 FlowerCloud，用其订阅链接重新导入。
❓ 国内网站也打不开了▾
多为「全球直连 / 漏网之鱼」被误改。请恢复默认：「全球直连」保持 DIRECT，「漏网之鱼」保持节点选择（macOS 详见第三步说明）。
❓ Mac 系统代理开关一打开就自动关闭▾
进入「设置」，点击虚拟网卡（TUN）旁的扳手图标安装服务模式，再开启 TUN 模式即可解决权限冲突。
❓ 登录机场官网收不到验证码▾
验证码发送到绑定的 Gmail 邮箱，请登录 Gmail 收件箱查看；注意验证码有几分钟时效，过期请在网页重新获取。
❓ iOS 的 App Store 搜不到 Shadowrocket▾
国区 App Store 没有该应用，需切换到境外 Apple ID（详见第二步「iOS」小节里的图文教程）。
